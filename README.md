# ROS2 Mecanum Trajectory Demo

A ROS2 package suite for holonomic robot control on mecanum wheels (tested on ROSbot XL). Includes forward kinematics, wheel velocity publisher, and eight-figure trajectory execution with feedback control.

## Dependencies
- ROS2 Humble or Iron
- Eigen3 (`sudo apt install libeigen3-dev`)
- tf2, nav_msgs, std_msgs (standard ROS2)
